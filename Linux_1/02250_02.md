## 指令

### 查詢網路資訊
```
ip address show
ip a s
```

### 中斷指令
```
CTRL + C
```

### 刪除指令
```
sudo kill -9 4637
```
- kill：刪除指令
- -9：強制執行
- 4637：process代碼

### ping
```
ping
ping -c 1 192.168.56.1
```
- -c 1：ping 一次
