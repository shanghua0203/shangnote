### 權限
```
rwx rwx rwx
[擁有者][群組][其他人]
```
- r(read)：可讀
- w(wirte)：可寫
- x(execution)：可執行
- 
```
rws rwx rwx
```
- s：可以讓使用者的權限臨時提升為root

```
------4   0   0
|   rws rwx rwx
|   111 111 111
4   7   7   7
```
```
------4   2   1
|   rws --s --t
|   111 001 000
7   7   1   0
```

## 指令介紹

### du
##### 查詢路徑下所有檔案資料夾大小
```
du <路徑名稱>
du -sh <路徑名稱>
```
- -s：sum
- -h：檔案大小會以K、G、M簡化

### df
##### 查看硬碟分割區情況

### find
```
find ./home/user -name "123.txt"
```
find [範圍] [條件] [動作]

#### 依照名稱
```
find . -name "file.txt"
find . -iname "FiLe.Txt"
```
- iname：不分大小寫
- .：當前目錄
- ..:上一個目錄
  
#### type   
```
find . -type f -name "aa*"  //檔案
find . -type d -name "aa*"  //資料夾
```
- f：檔案
- d：目錄
- 
#### -size
```
find . -size +10M
```
找出超過10MB的檔案
